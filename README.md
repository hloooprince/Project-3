# 🌦️ Weather Monitoring & Alert System

## 📌 Overview

The Weather Monitoring & Alert System is a simple Java console application developed using Object-Oriented Programming (OOP) concepts. It accepts temperature and humidity values from the user, analyzes the weather conditions, and generates alerts when values exceed predefined thresholds.

## 🚀 Features

* Input temperature and humidity data
* Detect high temperature conditions
* Detect low temperature conditions
* Detect high humidity conditions
* Display weather status report
* Demonstrates OOP concepts using classes and objects

## 🛠 Technologies Used

* Java
* OOP (Classes & Objects)
* Scanner Class
* Console-Based Interface

## 📂 Project Structure

```text
Weather.java
│
├── WeatherData Class
└── Weather Class (Main Method)
```

## ⚙️ Alert Conditions

| Condition          | Alert                  |
| ------------------ | ---------------------- |
| Temperature > 40°C | High Temperature Alert |
| Temperature < 10°C | Low Temperature Alert  |
| Humidity > 80%     | High Humidity Alert    |

## ▶️ How to Run

### Compile

```bash
javac Weather.java
```

### Execute

```bash
java Weather
```
